Downloader Studio

🚀 Démarrage Rapide - Downloader Studio

Commencez à utiliser Downloader Studio en 5 minutes !

Installation

Utilisateurs Windows

  1. Téléchargez l'installateur depuis les Releases
  2. Exécutez DownloaderStudio_Setup.exe
  3. Suivez l'assistant d'installation
  4. Lancez depuis le Menu Démarrer ou le Bureau

Développeurs

# Clonez le dépôt
git clone https://github.com/batsave/downloader-studio.git
cd downloader-studio

# Installez les dépendances
pip install -r requirements.txt

# Lancez l'application
python main.py

Utilisation de Base

1️⃣ Rechercher du Contenu

  • Allez à l'onglet Recherche
  • Entrez un URL ou une requête
  • Sélectionnez le format et la qualité
  • Cliquez Télécharger

2️⃣ Gérer la File d'Attente

  • Allez à l'onglet File d'attente
  • Voyez tous les téléchargements en attente
  • Cliquez Lancer la file pour commencer

3️⃣ Configurer l'Application

  • Allez à Paramètres
  • Choisissez la langue (FR, EN, DE, ES)
  • Sélectionnez le dossier de sortie
  • Configurez les formats par défaut

🌍 Changer la Langue

  1. Paramètres → Apparence → Thème
  2. Choisissez votre langue:
  • 🇫🇷 Français
  • 🇬🇧 English
  • 🇩🇪 Deutsch
  • 🇪🇸 Español

📦 Mode Minimaliste

Pressez Ctrl+M pour ouvrir une petite fenêtre flottante pour les téléchargements rapides.

🎨 Thèmes

  • Sombre (par défaut) - Parfait pour une utilisation prolongée
  • Clair - Idéal pour les environnements bien éclairés

📋 Formats Supportés

Vidéo:

  • MP4 (Full HD 1080p)
  • WebM

Audio:

  • MP3 (320kbps)
  • WAV
  • M4A

🔗 Sources Supportées

  • YouTube
  • SoundCloud

⚙️ Dossier de Sortie

Les fichiers téléchargés vont dans le dossier configuré (par défaut: %USERPROFILE%\Downloads).

🆘 Dépannage

L'application ne démarre pas?

  • Vérifiez que Python 3.8+ est installé
  • Réinstallez les dépendances: pip install -r requirements.txt

Les icônes ne s'affichent pas?

  • Vérifiez que le dossier assets/ existe
  • Relancez l'application

Problème avec un téléchargement?

  • Consultez l'onglet Logs
  • Vérifiez votre connexion internet
