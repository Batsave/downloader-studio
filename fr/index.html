Documentation Downloader Studio

Bienvenue dans la documentation de Downloader Studio. Cette section regroupe les guides utiles pour installer, utiliser, construire et comprendre l'application.

Guides

  • Démarrage rapide - installer l'app, la lancer et faire un premier téléchargement.
  • Guide de construction - créer l'exécutable PyInstaller puis l'installateur Windows.
  • Architecture - comprendre l'interface PyQt, le moteur de téléchargement et les traductions.
  • À propos - objectifs du projet, fonctionnalités et limites d'utilisation.

Parcours conseillé

  1. Installez les dépendances avec pip install -r requirements.txt.
  2. Lancez l'application avec python main.py.
  3. Réglez la langue, le thème, les formats, la qualité et le dossier de sortie.
  4. Recherchez un média ou collez une URL depuis la page Recherche.
  5. Ajoutez les résultats à la file ou téléchargez un élément immédiatement.

Valeurs par défaut

  • Dossier de sortie: %USERPROFILE%\Downloads
  • Thème: sombre
  • Langue: français
  • Formats: MP3, MP4, WAV, M4A
