# 📖 About - Downloader Studio

## What is Downloader Studio?

**Downloader Studio** is a modern and professional desktop application for downloading media from multiple sources with complete multilingual support and customizable themes.

### Current Version
- **v2.11** - Stable Release
- **License**: MIT (Open Source)
- **Platform**: Windows 10/11
- **Python**: 3.8+

## Main Features

### 🎥 Multi-Source Downloads
- YouTube
- SoundCloud
- And more coming soon...

### 🎵 Multiple Formats
**Video:**
- MP4 (Full HD 1080p)
- WebM

**Audio:**
- MP3 (320kbps)
- WAV
- M4A

### 🌍 4 Supported Languages
- 🇫🇷 Français
- 🇬🇧 English
- 🇩🇪 Deutsch
- 🇪🇸 Español

### 🎨 Customizable Themes
- Dark Mode (default)
- Light Mode

### 📦 Usage Modes
- **Normal Mode** - Full interface
- **Minimalist Mode** - Compact window (Ctrl+M)

### ⚙️ Advanced Configuration
- Output folder selection
- Default quality selection
- Enabled sources management

## Architecture

The application is built with:
- **PyQt5** - User interface
- **yt-dlp** - Media downloading
- **Python 3.8+** - Programming language

Modern structure:
- Clear separation of concerns
- Code organized into logical packages
- All imports using absolute paths
- Compatible with PyInstaller and Inno Setup

## Project History

Downloader Studio was born from the need for a simple yet powerful application to download videos and music quickly and efficiently.

The project has grown to become a professional solution with:
- Modern and clean interface
- Complete multilingual support
- Professional build pipeline
- Easy distribution to users
