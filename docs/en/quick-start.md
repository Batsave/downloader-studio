# 🚀 Quick Start - Downloader Studio

Get started using Downloader Studio in 5 minutes!

## Installation

### Windows Users

1. **Download the installer** from [Releases](https://github.com/batsave/downloader-studio/releases)
2. **Run** `DownloaderStudio_Setup.exe`
3. **Follow the installation wizard**
4. **Launch** from Start Menu or Desktop

### Developers

```bash
# Clone the repository
git clone https://github.com/batsave/downloader-studio.git
cd downloader-studio

# Install dependencies
pip install -r requirements.txt

# Launch the application
python main.py
```

## Basic Usage

### 1️⃣ Search for Content

- Go to the **Search** tab
- Enter a URL or search query
- Select format and quality
- Click **Download**

### 2️⃣ Manage Queue

- Go to the **Queue** tab
- View all pending downloads
- Click **Start Queue** to begin

### 3️⃣ Configure Application

- Go to **Settings**
- Choose language (FR, EN, DE, ES)
- Select output folder
- Configure default formats

## 🌍 Change Language

1. **Settings** → **Appearance** → **Theme**
2. Choose your language:
   - 🇫🇷 Français
   - 🇬🇧 English
   - 🇩🇪 Deutsch
   - 🇪🇸 Español

## 📦 Minimalist Mode

Press **Ctrl+M** to open a small floating window for quick downloads.

## 🎨 Themes

- **Dark** (default) - Perfect for extended use
- **Light** - Ideal for well-lit environments

## 📋 Supported Formats

**Video:**
- MP4 (Full HD 1080p)
- WebM

**Audio:**
- MP3 (320kbps)
- WAV
- M4A

## 🔗 Supported Sources

- YouTube
- SoundCloud

## ⚙️ Output Folder

Downloaded files go to the configured folder (default: `%USERPROFILE%\Downloads`).

## 🆘 Troubleshooting

**Application won't start?**
- Verify that Python 3.8+ is installed
- Reinstall dependencies: `pip install -r requirements.txt`

**Icons not displaying?**
- Check that the `assets/` folder exists
- Restart the application

**Download issue?**
- Check the **Logs** tab
- Verify your internet connection
