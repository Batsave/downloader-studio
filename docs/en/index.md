## Common Workflow

1. Install dependencies with `pip install -r requirements.txt`.
2. Run the app with `python main.py`.
3. Choose language, theme, formats, quality, and output folder in Settings.
4. Search or paste a URL from the Search page.
5. Add results to the queue or download one item immediately.

## Defaults

- Output folder: `%USERPROFILE%\Downloads`
- Theme: dark
- Language: French
- Formats: MP3, MP4, WAV, M4A
