# 📖 À Propos - Downloader Studio

## Qu'est-ce que Downloader Studio?

**Downloader Studio** est une application desktop moderne et professionnelle pour télécharger des médias à partir de multiples sources avec support complet du multilingue et de thèmes personnalisés.

### Version Actuelle
- **v2.11** - Release Stable
- **Licence**: MIT (Open Source)
- **Plateforme**: Windows 10/11
- **Python**: 3.8+

## Fonctionnalités Principales

### 🎥 Téléchargement Multi-Source
- YouTube
- SoundCloud
- Et plus à venir...

### 🎵 Formats Multiples
**Vidéo:**
- MP4 (Full HD 1080p)
- WebM

**Audio:**
- MP3 (320kbps)
- WAV
- M4A

### 🌍 4 Langues Supportées
- 🇫🇷 Français
- 🇬🇧 English
- 🇩🇪 Deutsch
- 🇪🇸 Español

### 🎨 Thèmes Personnalisables
- Mode Sombre (par défaut)
- Mode Clair

### 📦 Modes d'Utilisation
- **Mode Normal** - Interface complète
- **Mode Minimaliste** - Fenêtre compacte (Ctrl+M)

### ⚙️ Configuration Avancée
- Choix du dossier de sortie
- Sélection de la qualité par défaut
- Gestion des sources activées

## Architecture

L'application est construite avec:
- **PyQt5** - Interface utilisateur
- **yt-dlp** - Téléchargement de médias
- **Python 3.8+** - Langage de programmation

Structure moderne:
- Séparation claire des responsabilités
- Code organisé en packages logiques
- Tous les imports en chemins absolus
- Compatible PyInstaller et Inno Setup

## Histoire du Projet

Downloader Studio est né du besoin d'une application simple mais puissante pour télécharger des vidéos et musiques de manière rapide et efficace.

Le projet a grandi pour devenir une solution professionnelle avec:
- Interface moderne et épurée
- Support multilingue complet
- Build pipeline professionnel
- Distribution facile aux utilisateurs
