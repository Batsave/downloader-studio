# 🚀 Schnelleinstieg - Downloader Studio

Beginnen Sie in 5 Minuten mit Downloader Studio!

## Installation

### Windows-Benutzer

1. **Laden Sie das Installationsprogramm herunter** von [Releases](https://github.com/batsave/downloader-studio/releases)
2. **Führen Sie** `DownloaderStudio_Setup.exe` aus
3. **Folgen Sie dem Installationsassistenten**
4. **Starten Sie** über das Startmenü oder den Desktop

### Entwickler

```bash
# Repository klonen
git clone https://github.com/batsave/downloader-studio.git
cd downloader-studio

# Abhängigkeiten installieren
pip install -r requirements.txt

# Anwendung starten
python main.py
```

## Grundlegende Verwendung

### 1️⃣ Inhalte Suchen

- Gehen Sie zur Registerkarte **Suche**
- Geben Sie eine URL oder Suchanfrage ein
- Wählen Sie Format und Qualität aus
- Klicken Sie auf **Herunterladen**

### 2️⃣ Warteschlange Verwalten

- Gehen Sie zur Registerkarte **Warteschlange**
- Zeigen Sie alle ausstehenden Downloads an
- Klicken Sie auf **Warteschlange starten**, um zu beginnen

### 3️⃣ Anwendung Konfigurieren

- Gehen Sie zu **Einstellungen**
- Wählen Sie die Sprache (FR, EN, DE, ES)
- Wählen Sie das Ausgabeverzeichnis aus
- Konfigurieren Sie Standard-Formate

## 🌍 Sprache Ändern

1. **Einstellungen** → **Darstellung** → **Design**
2. Wählen Sie Ihre Sprache:
   - 🇫🇷 Français
   - 🇬🇧 English
   - 🇩🇪 Deutsch
   - 🇪🇸 Español

## 📦 Minimalistischer Modus

Drücken Sie **Strg+M**, um ein kleines schwebendes Fenster für schnelle Downloads zu öffnen.

## ⚙️ Ausgabeverzeichnis

Heruntergeladene Dateien gehen in das konfigurierte Verzeichnis (Standard: `%USERPROFILE%\Downloads`).

## 🆘 Fehlerbehebung

**Anwendung startet nicht?**
- Stellen Sie sicher, dass Python 3.8+ installiert ist
- Installieren Sie Abhängigkeiten neu: `pip install -r requirements.txt`

**Symbole werden nicht angezeigt?**
- Überprüfen Sie, ob der Ordner `assets/` vorhanden ist
- Starten Sie die Anwendung neu

**Download-Problem?**
- Überprüfen Sie die Registerkarte **Protokolle**
- Stellen Sie sicher, dass Ihre Internetverbindung aktiv ist
