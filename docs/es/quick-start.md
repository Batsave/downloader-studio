# 🚀 Inicio Rápido - Downloader Studio

¡Comience a usar Downloader Studio en 5 minutos!

## Instalación

### Usuarios de Windows

1. **Descargue el instalador** desde [Releases](https://github.com/batsave/downloader-studio/releases)
2. **Ejecute** `DownloaderStudio_Setup.exe`
3. **Siga el asistente de instalación**
4. **Inicie** desde el Menú Inicio o Escritorio

### Desarrolladores

```bash
# Clone el repositorio
git clone https://github.com/batsave/downloader-studio.git
cd downloader-studio

# Instale las dependencias
pip install -r requirements.txt

# Inicie la aplicación
python main.py
```

## Uso Básico

### 1️⃣ Buscar Contenido

- Vaya a la pestaña **Búsqueda**
- Ingrese una URL o consulta de búsqueda
- Seleccione formato y calidad
- Haga clic en **Descargar**

### 2️⃣ Gestionar Cola

- Vaya a la pestaña **Cola**
- Vea todas las descargas pendientes
- Haga clic en **Iniciar Cola** para comenzar

### 3️⃣ Configurar Aplicación

- Vaya a **Configuración**
- Elija idioma (FR, EN, DE, ES)
- Seleccione carpeta de salida
- Configure formatos predeterminados

## 🌍 Cambiar Idioma

1. **Configuración** → **Apariencia** → **Tema**
2. Elija su idioma:
   - 🇫🇷 Français
   - 🇬🇧 English
   - 🇩🇪 Deutsch
   - 🇪🇸 Español

## 📦 Modo Minimalista

Presione **Ctrl+M** para abrir una pequeña ventana flotante para descargas rápidas.

## 🎨 Temas

- **Oscuro** (predeterminado) - Perfecto para uso prolongado
- **Claro** - Ideal para ambientes bien iluminados

## 📋 Formatos Soportados

**Video:**
- MP4 (Full HD 1080p)
- WebM

**Audio:**
- MP3 (320kbps)
- WAV
- M4A

## 🔗 Fuentes Soportadas

- YouTube
- SoundCloud

## ⚙️ Carpeta de Salida

Los archivos descargados van a la carpeta configurada (predeterminado: `%USERPROFILE%\Downloads`).

## 🆘 Solución de Problemas

**¿La aplicación no inicia?**
- Verifique que Python 3.8+ esté instalado
- Reinstale las dependencias: `pip install -r requirements.txt`

**¿Los iconos no se muestran?**
- Verifique que la carpeta `assets/` exista
- Reinicie la aplicación

**¿Problema con la descarga?**
- Verifique la pestaña **Registros**
- Verifique su conexión a Internet
