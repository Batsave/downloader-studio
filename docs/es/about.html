Downloader Studio

📖 Acerca de - Downloader Studio

Qué es Downloader Studio?

Downloader Studio es una aplicación de escritorio moderna y profesional para descargar medios de múltiples fuentes con soporte completo de idiomas y temas personalizables.

Versión Actual

  • v2.12 - Versión Estable
  • Licencia: MIT (Código Abierto)
  • Plataforma: Windows 10/11
  • Python: 3.8+

Características Principales

🎥 Descargas Multi-Fuente

  • YouTube
  • SoundCloud
  • Y más próximamente...

🎵 Múltiples Formatos

Video:

  • MP4 (Full HD 1080p)
  • WebM

Audio:

  • MP3 (320kbps)
  • WAV
  • M4A

🌍 4 Idiomas Soportados

  • 🇫🇷 Français
  • 🇬🇧 English
  • 🇩🇪 Deutsch
  • 🇪🇸 Español

🎨 Temas Personalizables

  • Modo Oscuro (predeterminado)
  • Modo Claro

📦 Modos de Uso

  • Modo Normal - Interfaz completa
  • Modo Minimalista - Ventana compacta (Ctrl+M)

⚙️ Configuración Avanzada

  • Selección de carpeta de salida
  • Selección de calidad predeterminada
  • Gestión de fuentes habilitadas

Arquitectura

La aplicación está construida con:

  • PyQt5 - Interfaz de usuario
  • yt-dlp - Descarga de medios
  • Python 3.8+ - Lenguaje de programación

Estructura moderna:

  • Separación clara de responsabilidades
  • Código organizado en paquetes lógicos
  • Todos los imports con rutas absolutas
  • Compatible con PyInstaller e Inno Setup

Historia del Proyecto

Downloader Studio nació de la necesidad de una aplicación simple pero potente para descargar videos y música de manera rápida y eficiente.

El proyecto ha crecido para convertirse en una solución profesional con:

  • Interfaz moderna y limpia
  • Soporte multilingüe completo
  • Pipeline de compilación profesional
  • Distribución fácil a usuarios
