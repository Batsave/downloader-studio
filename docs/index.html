Flux recommandé

1. Installer

Utiliser l'installateur Windows ou lancer `python main.py` en développement.

2. Configurer

Choisir langue, thème, formats, qualité et dossier de sortie.

3. Télécharger

Rechercher ou coller une URL, puis ajouter à la file ou télécharger directement.

4. Publier

Construire `DownloaderStudio.exe`, compiler le `.iss`, tester l'installateur.

Par défaut, les fichiers sont écrits dans %USERPROFILE%\Downloads. Le fichier local downloader_settings.json est ignoré par Git.
